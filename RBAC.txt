ROLE BASED ACCESS RULES

ADMIN
-Has Global Privileges, Can Access all pages, make edits, create teacher accounts, enter marks, add students, edit settings, edit remarks, Generate Report Cards, view performance analytics, View all enrolled Students, View Class Lists for their Class


CLASS_TEACHER
- Can Enter Marks for all subjects in their class, view performance analytics, View Class Lists (Enrolled Students) for their Class, View Class Lists (Enrolled Students) for other Class and subjects they are assigned at least one subject.

SUBJECT_TEACHER
- Can Enter Marks for the assigned subject under a class, view performance analytics, View Class Lists (Enrolled Students) for the Class where they are assigned at least one subject, View Class Lists (Enrolled Students) Classes where they are assigned at least one subject.

A CLASS TEACHER CAN ALSO BE A SUBJECT TEACHER
